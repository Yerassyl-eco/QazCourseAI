# QazCourse AI

**AI-платформа для поиска лучших курсов и образовательных возможностей в Казахстане.**

Живой сайт: https://qazcourseai.lovable.app/

QazCourse AI — это «Google Maps для образования»: помогает пользователю понять, какая профессия ему подходит, какие навыки нужно прокачать и где именно в его городе можно этому научиться.

---

##  Возможности

-  **Онбординг** — выбор города и профессии (или AI-ассессмент, если пользователь ещё не определился)
- **AI-матчинг профессий** — 40 вопросов, анализ через Lovable AI Gateway, 3–5 рекомендованных профессий
-  **Каталог курсов** — фильтры по городу, формату, цене, уровню
-  **Персональные рекомендации** — на основе профиля и целевой профессии
-  **Сравнение курсов** — side-by-side анализ до 3 курсов
-  **Дашборд** — прогресс обучения, сохранённые курсы, next steps
-  **Мультиязычность** — Русский / Қазақша / English
-  **Аутентификация** — Email + Google OAuth (Lovable Cloud)

---

## 🛠 Технологии

- **Фреймворк:** TanStack Start v1 + React 19
- **Сборка:** Vite 7
- **Стили:** Tailwind CSS v4 + shadcn/ui
- **Backend:** Lovable Cloud (Supabase под капотом)
- **AI:** Lovable AI Gateway
- **Hosting:** Lovable / Cloudflare Workers
- **Пакетный менеджер:** Bun

##  Локальный запуск

```bash
# 1. Клонировать репозиторий
git clone <your-repo-url>
cd qazcourse-ai

# 2. Установить зависимости
bun install

# 3. Настроить переменные окружения
cp .env.example .env
# открой .env и подставь свои ключи (см. SECRETS.md)

# 4. Запустить дев-сервер
bun run dev
```

Откроется на `http://localhost:5173`.

---

##  Переменные окружения
Все ключи и инструкции — в [`SECRETS.md`](./SECRETS.md) и [`.env.example`](./.env.example).

**Главное правило:** `.env` никогда не коммитится. GitHub Secret Scanning блокирует пуш, если найдёт реальные ключи в коде.

## Как работает AI-матчинг

1. Пользователь проходит 40 вопросов (`src/lib/data/assessment.ts`)
2. Ответы отправляются в server function `aiMatch` (`src/lib/ai-match.functions.ts`)
3. Запрос идёт в Lovable AI Gateway с системным промптом
4. Модель возвращает топ-5 профессий с обоснованием
5. Пользователь выбирает одну → получает персональный набор курсов
